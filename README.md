# Anomaly Detection System

## Overview
This project detects anomalies in data using a Machine Learning model based on the Random Forest algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Data preprocessing
- Random Forest model training
- Anomaly prediction
- Confusion matrix visualization

## Project Structure
anomaly-detection-system
│
├── anomaly_alert_final.ipynb
├── requirements.txt
└── README.md

## Installation

1. Clone the repository
git clone https://github.com/your-username/anomaly-detection-system.git

2. Install required libraries
pip install -r requirements.txt

## Usage
Run the Jupyter notebook:

anomaly_alert_final.ipynb

The notebook will train the model and detect anomalies in the dataset.
