# Anomaly Detection System

## Overview
AI-based machine anomaly detection system using Random Forest, XG Boost, Linear regression and Isolation Forest.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features
- Data preprocessing
- traning models
- Anomaly prediction
- Confusion matrix visualization

## Project Structure
anomaly-detection-system
│
├── anomaly_alert_final.ipynb
├── requirements.txt
└── README.md

## Installation

1. Clone the repository
git clone https://github.com/Aanchal-tech123/anomaly-detection-system.git

2. Install required libraries
pip install -r requirements.txt

## Usage
Run the Jupyter notebook:

anomaly_alert_final.ipynb

The notebook will train the model and detect anomalies in the dataset.
